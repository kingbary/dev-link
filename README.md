# Devlink

Devlink is here to help you share your profiles with everyone!.

## Getting Started

First, clone the repository:

```bash
git clone https://github.com/kingbary/dev-link.git
```

Install dependencies:

```bash
pnpm install
```

Run the development server:

```bash
pnpm dev

```

